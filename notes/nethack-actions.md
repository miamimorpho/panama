move - Move in 8 directions (hjklyubn/arrow keys)
run - Run in a direction until something interesting
go - Move far in a direction (repeat move until blocked)
travel - Travel to a distant location automatically
jump - Jump over obstacles
teleport - Magical teleportation
climb_up - Go up stairs/ladders
climb_down - Go down stairs/ladders

Combat Actions

attack - Melee attack (automatic when moving into enemy)
fire - Fire quivered projectile
throw - Throw any item
fight - Force attack in a direction
kick - Kick in a direction
turn_undead - Special priest ability

Item Management

pick_up - Pick up items from ground
drop - Drop item from inventory
inventory - View inventory
apply - Use/activate a tool
wield - Equip a weapon
wear - Put on armor
put_on - Put on accessories (rings, amulets)
remove - Take off worn items
quiver - Ready ammunition for firing
swap_weapons - Exchange primary/secondary weapons

Consumption Actions

eat - Consume food
drink - Drink from container/fountain
read - Read scrolls, books, or engravings
zap - Use a wand
cast_spell - Cast memorized spell
invoke - Activate magical item power

Door/Container Actions

open - Open doors, containers
close - Close doors
lock - Lock doors/containers
force - Force open locked doors/containers
loot - Search containers/corpses

Information/Interface Actions

look - Examine nearby objects/monsters
search - Search for hidden doors/traps
rest - Pass time/heal (wait/noop)
show_inventory_type - Show specific inventory categories
count_gold - Count money
pay - Pay shopkeeper
chat - Talk to monsters

Special/Magical Actions

pray - Pray to your deity
offer - Make sacrifice on altar
dip - Dip items in liquids
rub - Rub items (lamps, stones)
untrap - Disarm traps
sit - Sit on throne/chair
ride - Mount/dismount steed
monster_ability - Use polymorphed form's special ability

Advanced Actions

engrave - Write on ground with tools
wipe - Clean face/eyes
adjust - Rearrange inventory letters
name - Name items/monsters
call - Give alternate names to item types
enhance - Improve weapon skills
two_weapon - Toggle dual-wielding mode

Meta/System Actions

save - Save game

redraw - Refresh screen

show_options - View/change game options
show_discoveries - List discovered item types
show_conduct - Check gameplay conduct challenges
